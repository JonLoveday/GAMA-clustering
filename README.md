# GAMA-clustering
Code and data for GAMA clustering analyses
corr.py: main python script
xi.c: pair-counting code

New version of the code (2017-aug-25).  Written and tested with python 3, but may also work with python 2.
This should only read standard GAMA DMUs using astropy tables.
gal_sample.py: classes and utilities for reading and manipulating galaxy samples
clust_util.py: classes and utilities for clustering analyses
util.py: generic utilities (as before)
group_clust.py: example code for group-galaxy cross-correlation
